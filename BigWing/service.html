Tecnologias

HTML

HTML, HyperText Markup Language, es la base estructural de la web. Sus etiquetas definen elementos, desde títulos hasta párrafos, dando forma y organización al contenido.

CSS

CSS, Cascading Style Sheets, añade estilo a las páginas web. Con simples reglas, controla colores, fuentes y diseño, mejorando la apariencia y la experiencia visual.

JS

JavaScript proporciona dinamismo y acción a las páginas web. Al responder a las acciones del usuario, como clics o desplazamientos, JS crea experiencias interactivas, haciendo que las páginas cobren vida.
